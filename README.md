# 🛸 HireRithm (하이어리즘)
> **AI 기반 맞춤형 채용 추천 및 보고서 자동화 플랫폼** <br>
> 구직자의 성향과 기업의 조직 문화를 정성적으로 분석하여 최적의 인재를 추천하는 채용 추천 웹 서비스

<br>

## 📌 Project Overview
기존 채용 플랫폼의 직무 위주 필터링과 낮은 인재 추천 신뢰도 문제를 해결하기 위해 기획된 웹 서비스입니다. 헤드헌터 및 인사 담당자에게는 객관적이고 신속한 인재 발굴 경험을, 구직자에게는 본인의 가치관과 부합하는 기업 매칭 경험을 제공합니다. 

* **개발 기간**: 2025.02 ~ 2025.11 
* **팀 구성**: 4인 (프론트/백엔드 - 김가연, 손호성, 송승규  , AI - 신의철)

<br>

<br>

## 💡 Motivation
기존의 채용 서비스들은 주로 직무 역량 중심의 '단순 키워드 필터링'에 의존하고 있어, 지원자의 가치관이나 성향, 그리고 기업이 가진 실제 조직 문화나 복지 같은 정성적인 요소를 충분히 반영하지 못한다는 한계가 있었습니다. 이로 인해 구직자는 본인의 성향과 잘 맞는 기업을 찾기 어렵고, 구인자 역시 단순 지표 중심의 추천 결과에 대해 낮은 신뢰도를 갖게 되는 문제가 발생했습니다.

또한 수백 명의 이력서를 일일이 검토하고 매칭 결과 보고서를 수작업으로 작성해야 하는 인사 담당자 및 헤드헌터들의 업무 부하를 줄일 수 있는 자동화 도구가 필요하다고 판단했습니다. '하이어리즘'은 이러한 문제를 해결하기 위해 다음과 같은 목표를 가지고 탄생했습니다:

* **정성적 데이터의 정량화**: 온라인 리뷰 크롤링과 AI 텍스트 마이닝을 통해 기업의 실제 문화를 수치화하여 매칭의 객관성을 확보합니다.
* **AI 기반의 신속하고 정확한 매칭**: 약 1,000명의 이력서를 30초 이내에 분석하여 편향되지 않은 최적의 후보자를 신속하게 도출합니다.
* **업무 프로세스 혁신**: 인재 추천부터 보고서 생성까지의 과정을 자동화하여 채용 담당자의 업무 효율성을 극대화합니다.

<br>

## 🎯 Key Features

### 1. 기업 이미지 기반 추천 (For 기업/헤드헌터)
* 잡플래닛 등 온라인 기업 리뷰 데이터를 크롤링하여 조직문화, 보상수준, 워라밸 등 정성적 요소를 정량화.
* 구직자의 '맞춤기업 TEST' 결과와 기업의 실제 이미지 점수를 비교 분석하여 일치도가 높은 순으로 후보자를 추천.

### 2. 강점 기반 후보자 추천 (For 기업/헤드헌터)
* AI를 활용해 구직자의 이력서와 기업의 요구 조건에서 핵심 강점 키워드를 추출하여 매칭.
* 1차 필터링 및 종합 분석을 통해 추천 사유를 함께 제공하여 추천 결과의 신뢰도 높이고자 함.

### 3. 업무 효율화: 추천 보고서 자동 생성
* 추천된 인재 리스트와 이력서 데이터를 바탕으로 PDF 형태의 추천 결과 보고서를 자동 생성하여 인사 담당자의 업무 소요 시간 90% 이상 단축.

<br>

## 🛠 Tech Stack

**Frontend**
* `React.js`
* `jsPDF` (보고서 자동화 처리)

**Backend**
* `Node.js`, `Express.js`
* `MongoDB`

**AI & Data**

* `Python`, `Selenium` (웹 크롤링)
* `PyTorch`, `KoBERT` (텍스트 마이닝 및 분류)
* `OpenAI GPT API` (자연어 처리 및 추천 로직)
